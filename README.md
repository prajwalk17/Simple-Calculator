# Simple-Calculator

A Python-based calculator with basic arithmetic operations and pay calculation with overtime.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Power/Exponent
- 💰 Pay Calculator (with 1.5x overtime after 40 hours)
